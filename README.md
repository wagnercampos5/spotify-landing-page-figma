# Projeto Landing Page Spotify - Curso FIC Web Design com Figma (IFSULDEMINAS)

Este repositório contém o projeto de design de interface (UX/UI) de uma Landing Page para o Spotify, desenvolvido como projeto prático para a conclusão do curso **Design de Websites com Figma** pelo **IFSULDEMINAS - Campus Machado**[cite: 1].

---

## 🚀 Sobre o Projeto

O objetivo deste projeto foi aplicar na prática conceitos fundamentais de design de interface, usabilidade e arquitetura de informação, criando uma Landing Page responsiva (nas versões Web e Mobile) focada em conversão para os planos do Spotify Premium[cite: 1]. O layout foi construído utilizando as melhores práticas do Figma, garantindo consistência, responsividade e organização das camadas do início ao fim[cite: 1].

---

## 🎨 Processo de Design

O desenvolvimento do projeto seguiu rigorosamente as seguintes etapas metodológicas[cite: 1]:

### 1. Guia de Estilo (Design System Inicial)
Definição e estruturação de variáveis globais de estilo de cores (baseadas na identidade Dark Mode oficial do Spotify) e tipografia para garantir consistência visual em todo o projeto[cite: 1].
> *Insira aqui o print do seu Guia de Estilos! (Arraste a imagem para este espaço)*

### 2. Wireframe (Esqueleto da Interface)
Planejamento estrutural das seções da página utilizando grades de layout (Grid de 12 colunas) e hierarquia de informação antes da aplicação dos elementos visuais definitivos[cite: 1].
> *Insira aqui o print do seu Wireframe! (Arraste a imagem para este espaço)*

### 3. Auto Layout & Componentização
Utilização intensiva da ferramenta de Auto Layout do Figma para simular o comportamento de caixas flexíveis, facilitando a futura transição para o código front-end (Flexbox no CSS)[cite: 1].

### 4. Design de Alta Fidelidade (UI Final)
Aplicação de imagens em alta definição, ícones vetoriais modernos e cores de contraste para gerar uma experiência de usuário imersiva, limpa e extremamente profissional[cite: 1].

## 📁 Organização das Páginas no Figma

Para manter o fluxo de trabalho organizado, o arquivo do Figma foi estruturado nas seguintes páginas[cite: 1]:
*   **Capa:** Identificação e organização do arquivo[cite: 1].
*   **Guia de Estilo:** Paletas de cores salvas e estilos de texto documentados[cite: 1].
*   **Wireframe:** Estruturas de baixa fidelidade para as versões Web e Mobile[cite: 1].
*   **Projeto:** Interface final de Alta Fidelidade responsiva pronta para apresentação[cite: 1].
